# 🎮Word Puzzle 

> Trò chơi giải ô chữ thời gian thực (Real-time Multiplayer) kết hợp cơ chế cạnh tranh - SPST môn học MLN122 - Kinh tế chính trị Mác - Lênin.

🌐 **Chơi thử phiên bản Web (Live Demo):** [word-puzzle-topaz.vercel.app](https://word-puzzle-topaz.vercel.app/)

---

## 🌟 Giới thiệu dự án
Word Puzzle Battle là Trò chơi giải ô chữ thời gian thực kết hợp cơ chế cạnh tranh, biến việc giải đố khô khan thành 1 trò chơi tương tác vui vẻ và kịch tính. 

## 🛠️ Công nghệ sử dụng (Tech Stack)
- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js, Socket.io (Real-time Engine)
- **Deployment:** Vercel (Frontend) & Render (Backend)

---

## 🚀 Hướng Dẫn Cài Đặt (Local Development)

Để chạy dự án này trên máy tính của bạn, hãy đảm bảo bạn đã cài đặt sẵn **[Node.js](https://nodejs.org/en/)**.

**Bước 1:** Clone repository này về máy của bạn.

**Bước 2:** Mở Terminal (Command Prompt) tại thư mục dự án và chạy lệnh sau để cài đặt các thư viện cần thiết:
```bash
npm install
```
**Bước 3:**: Khởi động server (khởi chạy cả Frontend và Backend):

```Bash
npm run dev
```
**Bước 4:**: Mở trình duyệt web và truy cập vào địa chỉ:
👉 http://localhost:3000/
